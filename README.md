# Oussama Lassoued — Portfolio Website

A premium, cinematic portfolio website showcasing UI/UX design expertise through immersive 3D graphics, sophisticated animations, and intentional typography.

## 🎯 Overview

This portfolio represents a high-end digital experience built with modern web technologies. It features:

- **Cinematic 3D/WebGL**: 800+ particle system with fog, distortion effects, and dynamic camera movement
- **Advanced Motion Design**: Scroll-based transitions, layered parallax depth, and staggered reveal animations
- **Monochrome Aesthetic**: Strict black and white palette for premium, editorial feel
- **Scene-Based Structure**: Five distinct visual scenes that flow like a narrative

## ✨ Key Features

### 1. Hero Scene
- Dramatic intro sequence with multi-layer animations
- 3D background with particles, geometric shapes, and atmospheric fog
- Blur, scale, and depth effects for cinematic entrance

### 2. Design Philosophy (Manifesto)
- Four core principles with editorial layout
- Scroll-scrubbing parallax animations
- Numbered hierarchy with hover interactions

### 3. Capabilities Grid
- Six capability cards with micro-interactions
- Staggered entrance animations
- Hover states with glow effects and corner accents

### 4. About/Profile
- Two-column layout with personal narrative
- Key strengths sidebar with animated reveals
- Decorative geometric elements

### 5. Contact (Finale)
- Grand closing with radial gradient atmosphere
- Enhanced CTAs with sweep animations
- Multiple contact options

## 🛠 Tech Stack

- **Framework**: Next.js 16.2.1 (App Router)
- **Language**: TypeScript 5.x
- **Styling**: Tailwind CSS 4.x
- **Animation**: GSAP 3.14.2 + ScrollTrigger
- **Smooth Scroll**: Lenis 1.3.21
- **3D/WebGL**: Three.js 0.183.2 + React Three Fiber + Drei
- **Motion**: Framer Motion 12.38.0

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Visit `http://localhost:3000` to view the site.
